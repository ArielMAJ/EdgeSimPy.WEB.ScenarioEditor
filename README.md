# EdgeSimPy Scenario Editor

A web frontend component of [EdgeSimPy.WEB.Sys](https://github.com/ArielMAJ/EdgeSimPy.WEB.Sys) for creating and configuring edge computing simulation scenarios.

## Quick Start

### Prerequisites

- Node.js v18+ and npm (or `bun`)

### Installation & Running

```sh
git clone https://github.com/ArielMAJ/EdgeSimPy.WEB.ScenarioEditor.git
cd EdgeSimPy.WEB.ScenarioEditor
bun install         # npm install
bun dev             # npm run dev
```

The app runs at `http://localhost:5173`

### Other Commands

```sh
bun run build       # npm run build     # Production build
bun run preview     # npm run preview   # Preview production build
bun run lint        # npm run lint      # Check code quality
```

## Tech Stack

React, TypeScript, Vite, shadcn-ui, Tailwind CSS, React Router, TanStack React Query
